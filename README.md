# CHAR Landing Page

A premium, animated landing page for the CHAR iOS app — an AI-powered health, nutrition, and productivity companion.

## Features

- **Dark-first design** with warm orange/teal accent palette matching the app
- **GSAP + ScrollTrigger** scroll-driven animations
- **Lenis** smooth scrolling
- **Responsive** — works perfectly on mobile, tablet, and desktop
- **Glass morphism** card effects with cursor-following glow
- **Interactive bento grid** feature showcase
- **Phone mockup** with CSS-only app screen simulations
- **FAQ accordion** with smooth transitions
- **Accessibility** — semantic HTML, ARIA labels, reduced-motion support

## Tech Stack

- Pure HTML, CSS, JavaScript (no framework)
- GSAP 3.12 + ScrollTrigger (CDN)
- Lenis Smooth Scroll (CDN)
- Google Fonts (Inter, JetBrains Mono)

## Development

Simply open `index.html` in a browser. No build step required.

For live reloading during development:
```bash
npx live-server
```

## Deployment

Hosted on GitHub Pages. Push to `main` branch to deploy.

## Screenshot Placeholders

The site includes CSS-only phone mockups. Replace with real screenshots by:
1. Adding images to `assets/mockups/`
2. Updating the `<img>` tags in `index.html`
